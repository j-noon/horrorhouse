About the Game

The idea of this game is to find your way through the rooms and escape. To escape the Horror House, you must find the token hidden within the game page and enter it into the token input box. There’s also an extra token for players who want to find all of Gumpie’s treasure. Along the way, you'll encounter educational content to explore and learn from.

How to Play

On-Screen Clicks

Click on the room images to search and navigate that room.

Keyboard and Mouse Advised

We strongly recommend using a keyboard and mouse for precise navigation and input. Default browser key binds can help. Click the purple text for new hints, or press the letter H!

Read Carefully

This is a puzzle game, everything you read might be a clue, so pay attention!

Use the Provided Hints

A hint button is provided. If you’re stuck, it’s there for a reason. Don’t need it? We’ll see!
